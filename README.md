# Задачка по выводу и фильтрации данных на Vue.js

 ## Задача:
Вывести массив данных о пользователях в виде таблицы и реализовать фильтрацию:
- по полу (все/мужчина/женщина)
                        И
- по возрасту (диапазон: старше Х/младше Х/от Х до Y).
